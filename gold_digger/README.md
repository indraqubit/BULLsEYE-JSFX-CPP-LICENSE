# GOLD DIGGER HUB

Folder `gold_digger/` adalah playbook reusable untuk pendekatan **“gold digging”**:

> gali pattern/hukum terbaik dari referensi, adopsi yang bernilai, buang baggage, lalu buktikan dengan contract + evidence.

## Isi

- `GOLD_DIGGING_PLAYBOOK.md`
  - Metode end-to-end (discovery → extraction → mapping → reimplementation → evidence gate).

- `GOLD_DIGGING_TEMPLATE.md`
  - Template siap pakai untuk dipakai di folder/fitur lain.

## Prinsip Ringkas

1. **Adopt laws/patterns, not baggage**
2. **Contract-first** (audio safety/correctness/lifecycle/perf/legal)
3. **Evidence-based completion** (jangan klaim done tanpa bukti)
